LimeLight
A private stage for everyone
Tell Me More

LimeLight is a deep learning-based
privacy tool that blurs faces in videos.



Features

Automated

All faces in videos are automatically blurred.

Selective

LimeLight only blurs faces that it does not recognize.

Secure

Raw videos are never stored on the servers once processing is complete.
